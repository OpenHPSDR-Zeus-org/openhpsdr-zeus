Hermes-Lite 2.x Wiki
====================

This Wiki is for community documentation of the Hermes-Lite 2.x. See the sidebar for links to various wiki pages. See the main [Hermes-Lite Web Page](http://www.hermeslite.com) for global links and details.

This is a community created and maintained Wiki. Please feel free to edit and contribute. All you need is a free github account.

# Getting Started

 1. Check out the [FAQ](FAQ) to see if this is a good project for you.
 1. Purchase a [preassembled kit](Group-Buy).
 1. Complete [final assembly](Final-Assembly) once your kit arrives.
 1. Connect to and understand all [front and back connections](External-Connections).
 1. Run [software](Software) to use the Hermes-Lite.
